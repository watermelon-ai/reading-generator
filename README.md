## Reading Generator

根据不同来源的文章生成高可阅读性的文章内容。

### Installation

```shell
pip install -r requirements.txt
```

### 生成

将要生成的内容放到当前目录的 `input.txt` 文件中，内容输出到 `output.txt` 文件。

### 执行

```shell
python -m examples.generate_twitter
```
